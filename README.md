# SwooshRGB
PWM RGB led strip over sACN (E1.31, DMX512) esp-idf

# Build and flash

## Build
```bash
idf.py build
```

## Flash
```bash
idf.py flash
```

# Serial Monitor
## Monitor
```bash
idf.py monitor
```

## Close monitor
Press <kbd>CTRL</kbd>+<kbd>]</kbd> to exit.

## Build and flash within monitor
Press <kbd>CTRL</kbd>+<kbd>T</kbd> then <kbd>A</kbd> to build and flash.
